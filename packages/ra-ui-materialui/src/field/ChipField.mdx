# ChipField

Displays a value inside a ["Chip"](http://www.material-ui.com/#/components/chip), which is Material UI's term for a label.

<Playground>
    <SimpleShowLayout resource="post" record={{ category: 'technology' }} style={{ display:'flex', justifyContent: 'space-around' }}>
        <ChipField source="category"  />
        <ChipField source="empty" />
    </SimpleShowLayout>
</Playground>

This field type is especially useful for lists, either coming from an Array value, or from a one to many relationship, e.g. to display a list tags of a given post:

<Playground>
    <SimpleShowLayout resource="post" record={{ id: 123, categories: [
        { name: 'technology' },
        { name: 'mobile' },
        { name: 'review' }
    ] }}>
        <ArrayField source="categories">
            <SingleFieldList>
                <ChipField source="name" />
            </SingleFieldList>
        </ArrayField>
    </SimpleShowLayout>
</Playground>

## Props

<PropsTable props={{
    addLabel: { 
        type: 'boolean',
        description: 'Add a label when used in a form',
        default: true
    },
    className: {
        type: 'string',
        description: 'A custom class name to be applied to the main element'
    },
    cellClassName: {
        type: 'string',
        description: 'A custom class name to be applied to the cell containing the component when used in a Datagrid'
    },
    headerClassName: {
        type: 'string',
        description: 'A custom class name to be applied to the header of the column of the component when used in a Datagrid'
    },
    label: {
        type: 'string',
        description: 'The field label. Will be translated by the i18n provider.',
        default: 'The humanized source (e.g. post_id => Post id)'
    },
    record: { 
        type: 'object',
        description: 'The curent record. Injected by react-admin.',
    },
    sortBy: { 
        type: 'string',
        description: 'The name of the field to use for sorting when the user clicks on the field header name.',
        default: 'The source'
    },
    source: { 
        type: 'string',
        description: 'The path to the value in the record. Can use dot notation (e.g. "author.name") when the record has sub objects.',
        required: 'true'
    }
}} />

## Examples

<Playground>
    <WithStyles styles={{ dark: { background: '#ccffe3' }}}>
    {({ classes }) => 
        <SimpleShowLayout resource="post" record={{ category: 'technology' }} style={{ display:'flex', justifyContent: 'space-around' }}>
            <ChipField source="category" label="Cat" />
            <ChipField source="category" addLabel={false} />
            <ChipField source="category" className={classes.dark} />
        </SimpleShowLayout>
    }
    </WithStyles>
</Playground>

Inside a `<Datagrid>`, you can benefit from the `headerClassName` and `cellClassName` props to style the filed header or cell.

<Playground>
<WithStyles styles={{ green: { background: '#ccffe3' }, yellow: { background: '#ffc' }}}>
    {({ classes }) => 
        <Datagrid resource="post" ids={[1, 2]} data={{
            1: { category: 'technology', subcategory: 'mobile' },
            2: { category: 'lifestyle', subcategory: 'deco' }
        }}>
            <ChipField source="category" headerClassName={classes.green} />
            <ChipField source="subcategory" cellClassName={classes.yellow} />
            <ChipField source="empty" />
        </Datagrid>
    }
    </WithStyles>
</Playground>

## Customizing the material-ui Chip

Additional props are passed down to the material-ui [`<Chip>`](https://material-ui.com/api/chip) component.

<Playground>  
    <SimpleShowLayout resource="post" record={{ category: 'technology' }} style={{ display:'flex', justifyContent: 'space-around' }}>
        <ChipField source="category" label="Clickable" onClick={() => alert('clicked')} />
        <ChipField source="category" label="Deletable" onDelete={() => alert('deleted')} />
        <ChipField source="category" label="With avatar" avatar={<Avatar>FZ</Avatar>} />
    </SimpleShowLayout>
</Playground>

## Injected Props

When used inside a `<Datagrid>` or `<CardActions>`, react-admin injects the current `record`, which is necessary to get the data.

<Playground>
    <ChipField source="category" record={{ id: 123, category: "technology" }} />
</Playground>
